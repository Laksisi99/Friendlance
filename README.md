# Friendlance
Simple Social Media Website created by Using PHP,HTML,and CSS

Features
- Sign UP
- Login
- Edit Profile
- Add post
- Like Post
- Comment Post
- Delete Post
- Add profile/cover images
- Edit profile/cover images
- Follow / Unfollow friends
- See others Posts
- Search users
- Delete/Edit/like comments

More updates will be added in the future.
